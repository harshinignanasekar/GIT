PROJECT SETUP 
Once you clone the repo, pleasd use npm install command to install dependencies.

As we move forward in the project, we will add more dependencies, so please remember the following point.  
Whenever you take pull and see changes in package.json, please use npm install command again
to install any new dependencies.

GIT GUIDELINESS

Please work on your own branch and drop a message whenever you do a push.

To change branches, you need to use this command
    git fetch origin
    git checkout branch_name
